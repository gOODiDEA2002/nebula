# Nebula Starter Service - 未来规划

## 版本规划

### 当前版本：2.0.1-SNAPSHOT

**功能特性**：

- HTTP RPC调用
- gRPC支持
- Nacos服务发现
- RabbitMQ消息队列
- Redis分布式锁

### v2.1.0 - 性能优化

**目标**：优化RPC性能和服务间通信效率。

**计划功能**：

1. **RPC性能优化**
   - 连接池优化
   - 请求批量处理
   - 响应压缩
   - 目标：RPC调用延迟降低30%

2. **服务发现优化**
   - 服务缓存机制
   - 健康检查优化
   - 负载均衡策略优化

3. **消息队列优化**
   - 消息批量处理
   - 消费者性能优化
   - 死信队列处理优化

**里程碑**：

- [ ] 完成性能基准测试（2025-12）
- [ ] 完成RPC优化（2026-01）
- [ ] 完成服务发现优化（2026-02）

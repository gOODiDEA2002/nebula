关于 Nebula Framework

Nebula 是一个现代化的 Java 后端框架，基于 Spring Boot 3.x 和 Java 21 构建， 提供企业级应用开发的完整解决方案。

核心特性

  • 现代化技术栈: Java 21 + Spring Boot 3.x + Maven
  • 模块化架构: 基于 DDD 原则的清晰模块划分
  • RPC 通信: HTTP RPC 和 gRPC 支持
  • 服务发现: Nacos 集成
  • 多级缓存: 本地 + 分布式缓存
  • 任务调度: XXL-Job 集成

本示例项目

这是一个简单的 Web 应用示例，演示了 Nebula Framework 的 Web 开发能力。

技术栈

  • nebula-starter-web
  • Spring Boot Web
  • Thymeleaf 模板引擎
  • Spring Boot Actuator

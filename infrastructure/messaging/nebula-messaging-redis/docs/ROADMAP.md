# Nebula Messaging Redis - 发展路线

## 当前状态

- 已支持 Redis Pub/Sub 与 Redis Stream 双模式
- 支持注解驱动的消息处理器
- 内置 JSON 序列化与自动配置

## 已知限制

1. Stream 高级特性（如消费者偏移量追踪）尚未提供可视化工具支持。
2. Pub/Sub 不保证消息持久化，需业务侧兜底。

## 计划特性

- 更完善的 Stream 消费监控与重试策略
- 支持多序列化策略的可插拔扩展
- 与监控体系联动的消息指标统计

## 版本规划

- v2.0.x：完善文档与示例，补齐测试覆盖
- v2.1.x：增强 Stream 消费者组管理能力
